# Security Policy

## Supported Versions
Currently, only the `main` branch and the latest tagged release are supported with security updates.

## Reporting a Vulnerability
Please do **not** report security vulnerabilities through public GitHub issues. 

Instead, please email **erickmontes555@gmail.com**. Include a description of the vulnerability and steps to reproduce it. We will try to acknowledge your email within 48 hours and provide a timeline for a fix.
